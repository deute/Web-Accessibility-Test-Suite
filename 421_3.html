4.2.1(웹 애플리케이션 접근성 준수) 콘텐츠에 포함된 웹 애플리케이션은 접근성이 있어야 한다.

자바스크립트 접근성(pseudo자바스크립트 사용 여부)

검수요소

검수대상
링크
검증 도구
PAJET

평가 순서

  1. 웹 페이지에 접속한다.
  2. 웹 페이지에 서식 묶음이 있는지 확인한다.
  3. PAJET을 실행한다.
  4. PAJET의 항목중 "4.2.1 웹 어플리케이션 접근성 준수(pseudo자바스크립트 사용 여부)"부분을 확인한다.
