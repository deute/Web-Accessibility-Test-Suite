4.2.1(웹 애플리케이션 접근성 준수) 콘텐츠에 포함된 웹 애플리케이션은 접근성이 있어야 한다.

자바스크립트 접근성(서식사용)

검수요소

검수대상
서식
검증 도구
PAJET, IE Developer toolbar

평가 순서

  1. 웹 페이지에 접속한다.
  2. 웹 페이지에 서식 묶음이 있는지 확인한다.
  3. PAJET을 실행한다.
  4. PAJET의 항목중 "4.2.1 웹 어플리케이션 접근성 준수(서식의 올바른 사용)"부분을 확인한다.
  5. 자바스크립트를 비활성화한다(IE developer toobar 툴바 > 사용안함 > 스크립트)
  6. 서식을 사용한다.

평가기준

  • PAJET에서 모든 form에 action과 submit버튼이 선언되어 있어야 한다.
  • 자바스크립트를 사용할수없는 환경에서도 서식을 사용하는데 무리가 없어야한다.
