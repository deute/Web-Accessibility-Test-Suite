3.4.2(오류 정정) 입력 오류를 정정할 수 있는 방법을 제공해야 한다.

검수요소

검수대상
페이지 전체
검증 도구
사용 평가

평가 순서

  1. 웹 페이지를 실행한다.
  2. 서식이 있는지 확인한다.
  3. 서식에 값을 입력하지 않은채 전송을 실행한다.
  4. 서식에 잘못된 형식의 값을 입력하고 전송을 실행한다.

평가기준

  • "값이 입력되지 않았다"또는 "잘못된 값이 입력되었다"는 메세지가 뜨면서 문제의 항목으로 이동되는지 확인한다.
