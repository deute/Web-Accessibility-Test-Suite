3.3.1(콘텐츠의 선형화) 콘텐츠는 논리적인 순서로 제공해야 한다.

검수요소

검수대상
html
검증 도구
IE developer Toolbar

평가 순서

  1. IE8에서 웹 페이지를 연다.
  2. IE developer Toolbar를 실행한다.(f12)
  3. 상단툴바 > 사용안함> CSS 를 클릭하여 CSS 를 비활성화한다.

평가기준

  • CSS가 비활성화 되어도 화면을 이해하는데 문제가없어야한다.
  • CSS가 비활성화 되어도 콘텐츠의 순서가 논리적으로 구성 되어있어야한다..
