4.2.1(웹 애플리케이션 접근성 준수) 콘텐츠에 포함된 웹 애플리케이션은 접근성이 있어야 한다.

자바스크립트 접근성(onclick 링크의 사용)

검수요소

검수대상
서식
검증 도구
PAJET

평가 순서

  1. 웹 페이지에 접속한다.
  2. 웹 페이지에 서식 묶음이 있는지 확인한다.
  3. PAJET을 실행한다.
  4. PAJET의 항목중 "4.2.1 웹 어플리케이션 접근성 준수(onclick 사용시 대체 URL체크)"부분을 확인한다.

평가기준

  • 링크,팝업으로 동작하는 onclick 은 href 에 이동할 페이지 값이 선언되어야한다.
