1.1.1 (적절한 대체 텍스트 제공) 텍스트 아닌 콘텐츠는 그 의미나 용도를 이해할 수 있도록 대체 텍스트를 제공해야 한다.

플래시 콘텐츠의 대체 콘텐츠

검수요소

검수대상
object, embed
검증 도구
화면확인 소스확인

평가 순서

  1. 검증할 웹페이지를 로드한다.
  2. 플래시가 사용된 부분을 찾는다.
  3. 페이지에서 플래시를 적용한 코드부분을 찾는다.

평가기준

  • Object 태그안에 대체 콘텐츠를 포함하는지 확인한다.
  • 위의 조건을 충족하지 않을경우, 플래시 콘텐츠의 내용을 동일하게 보여주는 콘텐츠를 따로 구성해도 문제는 없음
  • 플래시 설치안내는 대체 콘텐츠가 안됨
